# vue-vuetify-filters

## Настройка проекта

```
npm install
```

### Компиляция и горячая перезагрузка для разработки

```
npm run serve
```

### Компиляция и минификация для продакшн

```
npm run build
```

### Линтинг и исправление файлов

```
npm run lint
```

### Настройка конфигурации

Посмотрите [Configuration Reference](https://cli.vuejs.org/config/).
